# 📋 작업 브리프

**원 명령:** 개발팀이 간단한 비밀번호 강도 체크 HTML 도구를 만들고,
보안팀이 그 도구의 보안 취약점을 점검하고,
홍보팀이 해당 서비스 소개 문서를 작성해줘.

## 요약
비밀번호 강도 체크 도구의 개발, 보안 취약점 점검, 그리고 서비스 소개 문서 작성을 순차적으로 진행합니다.

## 분배
- **🤖 지현**: 사용자가 입력한 비밀번호의 복잡도(길이, 특수문자, 숫자 등)를 실시간으로 분석하여 강도를 표시하는 HTML/JS 기반의 '비밀번호 강도 체크 도구'를 구현하십시오.
- **🌐 경수**: 지현님이 구현한 비밀번호 강도 체크 도구의 소스 코드를 분석하여, 클라이언트 측 검증 우회 가능성이나 XSS 등 웹 취약점이 있는지 점검하고 팀장님께 보고하십시오.
- **🤝 민준**: 제작된 비밀번호 강도 체크 도구의 주요 기능과 보안적 강점을 정리하여, 고객사 제안 시 활용할 수 있는 서비스 소개 문서를 작성하십시오.
