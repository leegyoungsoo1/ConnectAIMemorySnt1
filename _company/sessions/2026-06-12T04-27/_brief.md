# 📋 작업 브리프

**원 명령:** 준혁이랑 지현이한테 같이 시켜줘. 준혁은 샘플 날씨 데이터를 JSON으로 만들고 Python으로 읽어서 출력하는 스크립트 만들어줘. 지현은 그 JSON 데이터를 보여주는 간단한 HTML 페이지 만들어줘. 두 파일 다 실행까지 해줘.

## 요약
준혁(Backend)과 지현(Frontend)이 협업하여 샘플 날씨 데이터 JSON 생성, Python 출력 스크립트 작성 및 이를 시각화하는 HTML 페이지를 구현하고 실행합니다.

## 분배
- **⚙️ 준혁**: 샘플 날씨 데이터를 담은 'weather_data.json' 파일을 생성하고, 해당 파일을 읽어 콘솔에 출력하는 Python 스크립트를 작성하여 실행하십시오.
- **🤖 지현**: 준혁이 생성한 'weather_data.json' 데이터를 불러와 웹 브라우저에 가독성 있게 표시하는 간단한 HTML 페이지를 구현하고 실행하십시오.
