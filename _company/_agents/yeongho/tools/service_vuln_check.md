# 🔴 서비스 취약점 점검 (영호 담당 — 2단계)

열린 포트별 서비스 취약점을 자동 점검합니다. 1단계 포트 스캔 결과(scan_results.json)를 자동으로 읽어 실행됩니다.

⚠️ **반드시 서면 승인된 대상에만 사용할 것**

## 점검 항목

| 서비스 | 점검 내용 |
|--------|----------|
| HTTP/8080 | 서버 헤더 노출, 보안 헤더 누락, 위험 메서드, 민감 경로 |
| SMB(445) | SMBv1 활성화 (EternalBlue 위험) |
| RPC(135) | 엔드포인트 열거 가능 여부 |
| RDP(3389) | BlueKeep 패치·NLA 강제 여부 |
| SSH(22) | 인증 방식, 브루트포스 방어 |

## 실행 방법
```bash
# 1단계 결과 자동 사용
python service_vuln_check.py

# 직접 지정
python service_vuln_check.py --target 127.0.0.1 --ports 80,445,135
```
